Setup Guide: ABC Hospitals Management System
    Extract: Copy the ABC_Hospitals folder into your XAMPP htdocs directory.
    Start XAMPP: Open XAMPP Control Panel and start Apache and MySQL.

Database:
    Go to http://localhost/phpmyadmin/.
    Create a new database named hospital_db.
    Import the hospital_db.sql file provided inside the zip.

Access: Open http://localhost/ABC_Hospitals/ in your browser.
